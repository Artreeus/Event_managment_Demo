## 📚 Getting Started in 3 Steps

### Step 1: Install & Run (2 minutes)
```bash
pnpm install
pnpm dev
```

### Step 2: Load Demo Data (1 minute)
Visit http://localhost:3001/seed-db and click "Seed Database"

### Step 3: Test (5 minutes)
Log in with `customer1@example.com` / `Password123!`

**Total: 8 minutes to fully working application!**
